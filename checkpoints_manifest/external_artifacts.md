# External artifacts manifest

Large artifacts should be attached to GitHub Releases, Zenodo, OSF, or equivalent archival storage.

Record each artifact as:

```text
name:
description:
source/result:
sha256:
size:
storage_url:
required_for:
```

Required future artifacts:

- Cr2 CAS(12,12) final determinant and coefficient checkpoint.
- Cr2 CAS(12,12) Hamiltonian/cache provenance.
- Active rotation file with hash/tag `active_rotation:c65788197e7dc63e`.
- Seed-resolved PE-GADTRN checkpoints or compact manifests.
- DTRN-LoRA adapter run manifests.
